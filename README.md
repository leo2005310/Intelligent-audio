# Intelligent-audio
一款基于ARM开发板开发的智能音响
该项目分为嵌入式端+服务器端+QT客户端
嵌入式端音响部分基于mini2440开发板,移植了linux操作系统,包括u-boot,内核和根文件系统。应用层使用开源库mplayer作为音频解码工具,可以播放在线音乐。播放控制上采用父子孙三个进程,父进程用select监听用户指令，子进程阻塞等待音乐播放结束,孙进程负责启动mplayer解析在线音乐,开发板外接了语音识别模块,通过串口完成了数据的交互
服务器端使用了阿里云服务器，主要完成了两个功能:一是搭建在线音乐平台, 二是负责管理所有音响和app。服务器端使用c++开发,基于libevent高并发框架,可以同时处理多个音响设备和app。数据处理上使用了MYSQL数据库和链表,数据库存储app的登陆,绑定信息,链表应用于转发数据是查询使用
APP采用QT图形框架，一是显示音响当前的状态和歌曲信息，二是往服务器发送控制指令，包括切歌,调节音量，播放模式等等
